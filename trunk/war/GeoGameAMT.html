Welcome to GeoGame

Welcome to the GeoGame. You will be participating in a treasure-hunt game shortly, but first we would like you to complete a few preliminary tasks before you move on to the main game. Please follow the instructions below, and read any dialog boxes you are presented with. Do not navigate off of this page until the game is over, and you have logged out. Please note that the game will begin automatically once there are sufficient players. Don't worry if the memory test is cut short, it will not be counted against you.

We would like to begin with a short memory test. Here, you will be presented with a sequence of digits, and then you will be asked to type them into an entry box and hit a submit button. Hit the Start button to begin the task.
